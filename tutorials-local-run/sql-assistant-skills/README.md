# SQL Assistant — Skills

Build an agent that loads specialized skills progressively using on-demand context loading.

## What It Covers

- Skills pattern: agent loads tools/context on demand
- Progressive capability expansion during a session
- Context management for large tool sets
- On-demand vs. always-loaded tools tradeoffs

## Framework

LangGraph (multi-agent)

## Prerequisites

- Python + API key
- No cluster or cloud needed — runs locally
