# Data Analysis Agent

Build a data analysis agent that sends reports to Slack.

## What It Covers

- Data analysis with LLM reasoning
- Tool use for querying and summarizing data
- Slack integration for report delivery
- Scheduled or triggered analysis workflows

## Framework

Deep Agents (LangGraph)

## Prerequisites

- Python + API key
- Slack webhook or bot token
- No cluster or cloud needed — runs locally
