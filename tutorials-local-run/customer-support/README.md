# Customer Support — Handoffs

Build a customer support workflow where a single agent transitions between different states.

## What It Covers

- Agent state transitions (handoff pattern)
- Conditional edges based on conversation context
- Escalation paths (e.g., billing → human agent)
- Single agent handling multiple roles via state

## Framework

LangGraph (multi-agent)

## Prerequisites

- Python + API key
- No cluster or cloud needed — runs locally
