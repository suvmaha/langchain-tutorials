# Custom SQL Agent

Implement a SQL agent directly in LangGraph for maximum flexibility.

## What It Covers

- SQL agent as explicit LangGraph nodes
- Full control over the generate → execute → respond loop
- Error handling and retry logic as graph edges
- Comparison with LangChain's built-in SQL agent

## Framework

LangGraph

## Prerequisites

- Python + API key
- No cluster or cloud needed — runs locally
