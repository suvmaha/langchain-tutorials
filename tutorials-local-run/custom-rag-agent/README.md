# Custom RAG Agent

Build a RAG agent using LangGraph primitives for fine-grained control.

## What It Covers

- RAG pipeline implemented as a LangGraph graph
- Custom retrieval and generation nodes
- State management across retrieval steps
- When to use LangGraph over LangChain's built-in RAG

## Framework

LangGraph

## Prerequisites

- Python + API key
- No cluster or cloud needed — runs locally
