# Deep Research Agent

Build a multi-step web research agent with subagent delegation and strategic reflection.

## What It Covers

- Multi-step research planning
- Web search tool integration
- Subagent delegation for parallel research threads
- Reflection: agent evaluates its own findings and decides if more research is needed
- Final synthesis across all gathered information

## Framework

Deep Agents (LangGraph)

## Prerequisites

- Python + API key
- Web search API (e.g., Tavily)
- No cluster or cloud needed — runs locally
