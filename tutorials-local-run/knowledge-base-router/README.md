# Knowledge Base Router

Build a multi-source knowledge base that routes queries to specialized agents.

## What It Covers

- Router pattern: classify query → select agent
- Multiple specialized agents (e.g., docs, FAQ, database)
- Routing logic as a LangGraph conditional edge
- Combining results from different knowledge sources

## Framework

LangGraph (multi-agent)

## Prerequisites

- Python + API key
- No cluster or cloud needed — runs locally
